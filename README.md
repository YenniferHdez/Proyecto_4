# Proyecto_4.- Calculadora de notas UOC

Este proyecto surgió de una necesidad personal porque en la UOC(Universitat Oberta de Catalunya) no suelen trabajar con el sistema de puntuación del 1 al 10 al que estoy acostumbrada sino que, por ejemplo, puedo sacar en una prueba un 22/25. Como no quería perder mi tiempo haciendo reglas de tres, realicé este sencillo proyecto que hace la regla de tres por ti. 

Importante: No poner los números decimales con coma, sino con punto. Por ejemplo, no pongas 19,2, pon 19.2.
